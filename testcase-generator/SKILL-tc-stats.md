# TC 통계 스킬

<command-name>tc-stats</command-name>

생성된 테스트케이스의 통계 및 분포를 분석합니다.

## 트리거

- `/tc-stats`
- `/stats`

## 자동 트리거 조건

1. **검증 통과 후** (자동 실행)
2. **키워드 감지**:
   - "통계", "요약", "분포", "현황"
   - "몇 개?", "TC 수"

## 사용법

```
/tc-stats <Excel_파일_경로>
/tc-stats  # 가장 최근 생성된 TC 자동 선택
```

## 통계 항목

### 1. 전체 요약
- 총 TC 수
- 생성일시
- 소스 파일

### 2. 컴포넌트별 분포
- Button, Input, Popup 등 컴포넌트별 TC 수
- 비율 (%)

### 3. 테스트 유형별 분포
- 표시 확인, 기능 확인, Hover 확인 등
- 비율 (%)

### 4. Depth별 분포
- Depth 1 (대분류) 별 TC 분포
- Depth 2 (중분류) 별 TC 분포

## 상세 분석 옵션

```
/tc-stats --detail  # 각 컴포넌트별 상세 TC 목록
/tc-stats --export  # 통계를 별도 파일로 저장
```

### 상세 출력 예시

```
------------------------------------------------------------
Button TC 상세 (15개)
------------------------------------------------------------
1. IT_OP_BT_001: [Save] 버튼 클릭
2. IT_OP_BT_002: [Save] 버튼 Hover
3. IT_OP_BT_003: 저장 단축키 입력 (Ctrl+S)
4. IT_OP_BT_004: [Close] 버튼 클릭 - 작업내역 없음
5. IT_OP_BT_005: [Close] 버튼 클릭 - 작업내역 있음
...
```

## 활용 방안

1. **TC 커버리지 확인**: 특정 컴포넌트 TC 부족 여부 파악
2. **테스트 계획 수립**: 유형별 분포 기반 테스트 시간 산정
3. **품질 리포트**: 프로젝트 문서화에 활용
